# RFID Dashboard - Final UI (React + Vite + Material UI)
Versión final UI construida para presentación. Incluye:
- Material UI theme con paleta solicitada
- Sidebar responsivo y colapsable
- Dashboard con KPIs y gráficos (mock)
- Inventario con chips de estado y modal de detalle
- Eventos, Antenas, Historial y Reportes
- Servicios API mock en `src/services/api.js` para fácil sustitución por backend

## Ejecutar
1. npm install
2. npm run dev
3. Abrir http://localhost:5173/
